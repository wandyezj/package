# package

Simple Package Template

## Deploy

1. install
    > npm install

1. build
    > npm run build

1. test
    > npm run test

    - make sure all tests pacc

1. patch
    > npm version patch

1. update [versions](versions)

1. publish

    > npm publish

1. commit 
    > git commit -am "new version"

1. push
    > git push
